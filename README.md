# Infusion Drug Billing Certification Course

A comprehensive online certification course for medical billers learning infusion drug billing.

## Features

- **7 Learning Modules** covering all aspects of infusion billing
- **Interactive Quizzes** with 10-20 questions per module
- **95% Pass Rate Required** ensures thorough understanding
- **Unlimited Attempts** allows students to learn at their own pace
- **Progress Tracking** saves student progress locally
- **Printable Certificate** upon successful completion
- **Mobile Responsive** works on all devices

## Course Structure

1. **Module 1:** Introduction & Terminology
2. **Module 2:** Billing Unit Conversion
3. **Module 3:** The 5-Step Billing Process
4. **Module 4:** Drug-Specific Requirements
5. **Module 5:** Administration Codes & Modifiers
6. **Module 6:** Common Scenarios & Mistakes
7. **Final Exam:** Comprehensive certification exam

## How to Use

1. Visit the course website
2. Enter your full name
3. Complete each module in order
4. Pass each quiz with 95% or higher
5. Earn your certificate upon passing the final exam

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- LocalStorage for progress tracking

## Deployment

This course is deployed using GitHub Pages and requires no server-side processing. All data is stored locally in the student's browser.
